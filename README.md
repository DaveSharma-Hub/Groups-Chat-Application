# Groups-Chat-Application
Online personal and group chat application using HTML, CSS, JS, PHP,SQL and AJAX that allows users to create own groups, chat, make friends, check new notifications and edit functionality. Chat automatically updates using AJAX to allow for real-time chatting with other users collected from the database. Notification system uses an MVC model to update real-time notifications with the indicated number in red. Plus, the application takes into consideration some security concerns and therefore stores passwords as a hash using SHA-256, and prevents SQL injections via prepared statements.
